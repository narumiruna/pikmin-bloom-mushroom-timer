# pikmin-bloom-mushroom-timer

🍄 Pikmin Bloom 蘑菇計時器 - 多組倒數計時工具

## 🌐 線上使用

**網址：** https://narumiruna.github.io/pikmin-bloom-mushroom-timer/

## 📖 使用方法

1. **輸入時間**
   - 支援三種格式：`hh:mm:ss`（如 1:50:30）、`mm:ss`（如 5:10）、`ss`（如 30）
   - 可使用冒號或空白分隔：`1:50:30` 或 `1 50 30` 都可以
   - 系統會自動驗證格式並即時顯示解析結果

2. **設定重生與緩衝時間**
   - **重生時間（分鐘）**：蘑菇重生所需的時間
   - **緩衝時間（秒）**：提前提醒的時間
   - **重複鈴聲**：勾選後，時間到時會每 3 秒重複播放鈴聲，直到移除計時器
   - 最終計時 = 輸入時間 + 重生時間 - 緩衝時間

3. **管理計時器**
   - 點擊 ➕ **加入倒數** 新增計時器
   - 每組計時器有獨立的顏色標籤
   - 點擊 ✕ 按鈕可移除不需要的計時器（會自動停止重複鈴聲）

4. **計時提醒**
   - 剩餘 ≤ 15 秒：橘色警告並閃爍
   - 時間到：紅色高亮、播放和諧鈴聲（C 大調和弦）、瀏覽器通知
   - 如果勾選了「重複鈴聲」，會持續播放直到移除計時器
